# Thumbnail Hero Asset

This folder contains a standalone hero visual for project thumbnail use.

## File

- `hero.html` — static `1200x630` canvas built with HTML + inline Tailwind utility classes.

## Quick use

1. Open `hero.html` in a browser.
2. Capture/export the visible canvas as PNG.
3. Use the exported image as the project thumbnail (README/social preview).

## Notes

- Copy and claims are aligned with the repository scope: serverless/stateless crawler, 3-hour schedule, multi-supplier tracking, spread metric, and Google Chat/Telegram notifications.
- If you need a new style variant, duplicate `hero.html` and adjust only text/colors/layout while keeping the `1200x630` canvas.
